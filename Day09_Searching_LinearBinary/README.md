# Day 9 - Searching: Linear Search & Binary Search

## Problem
Find the index of a target value in a sorted array using binary search.

## Concepts Covered
- Linear search (O(n), any array)
- Binary search (O(log n), sorted array only)
- Overflow-safe mid calculation: low + (high-low)/2
- Search space reduction technique
